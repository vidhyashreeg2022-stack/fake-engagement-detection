# Fake Engagement Detection in Social Media
Detecting artificial engagement using behavioural, temporal, network, and linguistic analytics.

## Overview
This project identifies fake engagement generated by bots, automated scripts, and coordinated groups on social media platforms.  
Using synthetic behavioural data, the model distinguishes organic users from fake ones by analysing timing patterns, burst activity, interaction networks, and linguistic similarities.

## Dataset
- Type: Synthetic dataset  
- Size: 15,000 user activity records  
- Reason for synthetic data: Real social media behavioural logs are private and restricted  
- Features included:
  - Temporal behaviour features  
  - Network interaction features  
  - Linguistic behaviour features  

## Features Used
### Temporal
- avg_engagement_interval  
- timing_regularization  
- nighttime_activity_ratio  
- burst_engagement_count  

### Network
- unique_profiles_interacted  
- repeated_engagement_ratio  
- engagement_network_density  
- suspicious_group_overlap  

### Linguistic
- comment_similarity_score  
- keyword_bot_flag  
- language_switch_frequency  
- sentiment_pattern_stability  

### Labels
- authenticity_score  
- bot_probability  
- anomaly_type  
- flag_label  

## Feature Engineering
- Burst Ratio  
- Interaction Diversity Index  
- Repetition Score  
- Behaviour Stability Index  
- Composite Authenticity Score  

## Model Architecture
- Bot Classification → XGBoost Classifier  
- Authenticity Score Regression → Random Forest Regressor  
- Anomaly Detection → Isolation Forest  
- Coordinated Group Detection → Graph-Based Clustering  

## How to Run
### 1. Install dependencies
pip install -r requirements.txt

### 2. Run ML Pipeline
python fake_engagement_ml_pipeline.py

## Outputs
- Authenticity score  
- Bot probability  
- Behavioural anomaly flags  
- Visual plots (heatmap, importance, distributions, anomalies)
